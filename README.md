## 🚀 Featured Projects

### 🔹 AutoSense  
### *AI-Powered Predictive Maintenance Platform*

AutoSense is an intelligent predictive maintenance ecosystem designed for EVs, passenger vehicles, and heavy truck engines.  
The platform combines temporal deep learning, fleet analytics, independent ML inference APIs, and an in-app AI assistant called **AutoPilot** to help organizations monitor vehicle health and estimate Remaining Useful Life (RUL).

#### Highlights
- Temporal Deep Learning for truck engine risk prediction
- RUL estimation for EVs & conventional vehicles
- Fleet analytics dashboard with historical insights
- JWT-authenticated workflows for fleet/company users
- AutoPilot AI assistant integrated into the platform
- React frontend with protected routes & model health checks

#### Stack
`React` • `FastAPI` • `TensorFlow` • `PyTorch` • `JWT Auth` • `Docker`

🔗 [View Project](https://github.com/Gitansh16/AutoSense)

### 🔹 ANITA  
### *Advanced Novel Integration of a Tiered Aggregator for Credit Card Fraud Detection*

ANITA is a research-driven fraud detection framework focused on improving fraud prediction performance on highly imbalanced financial datasets using ensemble learning techniques.

The project evaluates multiple machine learning classifiers and combines them using advanced aggregation strategies to improve robustness, recall, and fraud detection accuracy.

#### Highlights
- Ensemble-based fraud detection architecture
- Voting & Stacking classifiers
- Comparative analysis of ML algorithms
- Optimized for imbalanced financial datasets
- Research paper accepted at **ITAI 2025**

#### Models Used
- Logistic Regression
- Random Forest
- SVM
- XGBoost

#### Evaluation Metrics
Accuracy • Precision • Recall • F1-Score • ROC-AUC

#### Stack
`Python` • `Scikit-Learn` • `XGBoost` • `Pandas` • `NumPy`

🔗 [View Research](https://github.com/Gitansh16)
🔗 [Research Paper](https://link.springer.com/chapter/10.1007/978-981-96-8687-2_27)
